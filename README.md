# 이음(理音) · 선부3동 지역사회보장협의체 나눔 매칭 앱

> 도움이 필요한 사람이 직접 나서지 않아도, 동네의 선의가 닿게 하는 앱

## 라이브 프리젠테이션

배포 사이트에서 바로 볼 수 있습니다 → **https://kevinsaem.github.io/ium/**

- ← → 방향키 · 클릭으로 슬라이드 넘기기
- `⌘P` (Ctrl+P) → PDF로 저장 가능 (16:9)

## 구성

- `index.html` — 6장짜리 선부3동 지역사회보장협의체 위원 제안 프리젠테이션
- `shots/` — 프로토타입 앱 화면 캡처 (역할별)
- `자료실/` — 기획 원본, 프로토타입 HTML, 원본 프리젠테이션 파일

## 기술 스택 (예정)

FastAPI · HTMX · Alpine.js · Tailwind CSS · Jinja2 · PostgreSQL
